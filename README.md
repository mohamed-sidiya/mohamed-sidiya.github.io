# Portfolio de Mohamed Abdellahi Sidiya

## Publication sur GitHub Pages

1. Ouvrir le dépôt `mohamed-sidiya/mohamed-sidiya.github.io`.
2. Ajouter tous les fichiers et dossiers contenus dans ce dossier à la racine du dépôt.
3. Faire un commit sur la branche `main`.
4. Aller dans **Settings → Pages**.
5. Dans **Build and deployment**, sélectionner **Deploy from a branch**.
6. Choisir la branche `main` et le dossier `/ (root)`.
7. Le site sera disponible à `https://mohamed-sidiya.github.io`.

Le CV est intégré dans `assets/CV_Mohamed_Abdellahi_Sidiya.pdf`.
